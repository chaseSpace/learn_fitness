# 学习健身（Learn Fitness）

## 为何要学习健身
社会要发展，科技要进步，GDP要增长，男性要养家庭，女性要爱美。

人活在世上，几乎不可能做到无欲无求。无论你是为了提升身体素质还是为了自身可持续发展，你都需要学习健身。

其实，大部分人都只是为了更好地生活而努力活着。你现在或许有很好的精力和赚钱能力，但三年后、五年后以及有家庭、有贷款后呢？你的工作是否不会损耗你的身体？
你的家庭是否不会消耗你的精力？除此之外你就没有别的梦想了吗？假设，我是说假设，你坚持健身了三年，能穿得一身挺拔西装，脱掉衣服至少有六块腹肌，
那么你走到任何场合还是否还会毫无存在感呢？**男人至死是少年，少年，健身吧！**

**健身磨炼心性**。无论你现在是自卑、做事三分钟热度、不甘心甘于平淡的自己等等，你也希望改变自己，但苦于没有人带动你，没有好的方法，
那么我推荐你开始健身吧！健身是一个长期的、持续性的过程，这个过程中你会体验到体重从刚开始的起起落落，逐渐变为掉秤十几斤甚至几十斤。
健身也是一门学问，不是无脑锻炼就可以达到目的的，所以等你坚持下来，掌握健身，掌握其中的知识，你自身的心性也会得到提升。

**不是八块腹肌才叫健身成功**。很多人会产生一个误解，认为坚持了几个月或一年的健身没有达到八块腹肌或者说体脂率没有掉的很多，就认为健身太难，无法坚持。
其实不然，倘若你真的坚持下来（至少每周都有3次以上的健身训练），不需要太久，坚持1个月你就能看到自己的体脂率变化（体重会掉的更明显）。以笔者为例（173cm/132斤），
在以2天跑步一次的频率坚持1个半月后，体重掉秤10斤（现保持121\~122斤），没有任何节食行为。贴图为证（早期体重由于称未到所以未记录）

当然，这里不是说就推荐跑步作为健身计划，只是举个实际案例作为参考。具体每个人的健身计划是根据个人情况可能不同的。

**壮硕的身体素质，与不讲道理的人讲道理**。你可以执行更专业的健身计划，比如使用哑铃、杠铃等专业器具，并购买蛋白粉肌酸等辅助健身饮食，帮助你打造更壮硕的身体。
虽然法律不推荐用👊🏻解决问题，但必须要承认的是，在遇到大多数生活中的小事的时候，比如遭遇插队、碰撞、路怒症等情况，**壮硕强健的身体有时能帮你做到不战而屈人之兵**。
但你若没有，就只能受欺负，这是没有任何办法的，这也是一种弱肉强食的弱化表现。

## 认识体脂率
体脂率是衡量人体肥胖的最佳标准，所以这里将大部分人的身体情况根据体脂率简单分为三类：体脂率低、体脂率标准和体脂率高。下面通过一张图来简单认识年龄与体脂率的对应关系：

<img src="./images/body_fat_ratio.gif" width="450">

其中，「偏瘦」可以认为是经常健身的人的体脂率范围，而大部分人都是在「正常」这个档位。**但是，如果你想要使身体更加健康，并且「穿衣显瘦，脱衣有肉」，那就尽量使自己进入偏瘦这个档位**。

体脂率的测量可以简单通过市面上￥50+的**体脂称**来测量，另外还可购买较专业的**体脂测量仪**（通常价格￥100+）进行测量。对于多数朋友，体脂称够用。

>注意，购买体重秤或体脂称的时候不要贪图便宜，经笔者实测，部分较便宜（￥10~30内）的称测量的体重数据在无进食的几小时内是波动的，不准确。一个测量不准确的称会严重影响你的健身心态。

## 饮食计划
请牢记：三分练，七分吃，十分睡。

### 早餐要点
- 把红薯、燕麦、玉米当主食；辅助豆浆/牛奶（尽量是脱脂牛奶）
- 还可以吃黑米芝麻糊（现榨可选配几颗核桃仁）
- 搭配富含维生素的水果吃更佳，如西红柿/圣女果/牛油果

### 中餐要点
- 主食尽量用糙米饭代替米饭面条（糙米饭需提前泡2h也可泡过夜，如长期吃最好1:1混合大米营养均衡，注意大米无需浸泡；煮前可加一勺白醋使米饭更松软）
- 多吃高蛋白低脂肪食物，如清蒸鸡/鱼/生蚝、白灼虾、酱牛肉、豆腐蒸虾仁等
- 多吃高膳食纤维食物，如蘑菇、紫菜等各类蔬菜
- 高淀粉食物不要做成菜和米饭一起吃，如土豆、藕、粉丝、红薯粉、山药、芋头、南瓜等，要么直接作为主食
- 减肥吃到5分饱，保持身材吃到7分饱

中餐食谱推荐（主要是减脂菜谱，可适当一周一顿欺骗餐）：
- 主食
    - 荞麦面/普通面条也可但控制分量（烹饪方式：煮熟后放入冷水5分钟，食用时倒掉冷水，面条更劲道更低热量；冷水不要直接用自来水，如没有冷开水可提前烧好开水+开水制成的冰块）
    - 糙米饭
- 素菜
    - 黄瓜炒鸡蛋、荷兰豆炒木耳、番茄炒菌菇、 包菜炒鸡蛋、青红尖椒炒花菜、青红尖椒炒煮鸡蛋、豆芽炒鸡蛋、苦瓜炒蛋、鸡蛋炒魔芋面/粉（加一点胡萝卜+青菜+火腿）
- 凉拌菜
    - 凉拌汁：蒜末+辣椒粉+葱花+香菜+小米辣+白芝麻+热油+一勺生抽+一勺醋+一勺蚝油，自行搭配
    - 木耳+平菇/金针菇+菠菜、黄瓜+虾仁+煮鸡蛋切块、凉拌莴笋、凉拌茄子、凉拌手撕鸡腿肉（去皮、加柠檬香菜）
- 荤菜
    - 芹菜炒牛肉、西蓝花炒虾仁、黄瓜/口蘑/西蓝花/娃娃菜/洋葱炒鸡胸肉（可加尖椒）、鸡胸肉沫蒸鸡蛋（15min）、大葱烧鸡胸/腿肉
- 汤类
    - 荷包蛋紫菜汤、番茄豆腐肉末汤、番茄鸡蛋虾仁汤、虾滑紫菜鸡蛋汤、番茄肥牛白菜汤、丝瓜牛肉平菇汤、豆腐青瓜虾仁汤、番茄金针菇鸡蛋汤
- 快手菜
    - 包菜鸡蛋卷（刷辣椒酱更有味）、糖醋荷包蛋

### 晚餐要点
- 晚餐必须吃，即使是减肥期间，否则降低代谢后使身体变成易胖体质
- 晚餐禁食豆浆、牛奶、水果（高糖），必须要有优质蛋白质（鱼虾、鸡胸肉/去皮鸡腿、牛肉），维生素（黄瓜冬瓜苦瓜海带木耳、菠菜生菜西红柿 ）
- 晚餐禁食营养高的煲汤（脂肪含量高）
- 若本身代谢高，就再加点主食，如紫薯、红薯、玉米、山药、土豆
- 尽量七点前吃
- 晚餐1h后若要运动，若是增肌期间最好运动后增加以蛋白质为主的加餐（运动指40min以上的运动），睡觉前2h不建议运动。

优质晚餐会帮助你掉秤，可晚餐前称重，第二天早上空腹称重比较。原理是当进食低脂低卡类食物时，这些食物进入身体后将全部被消化而没有多余能量转化为脂肪，并且人体消化食物也需要能量，如此一来可形成热量缺口，以达到减脂效果。

晚餐食谱推荐：
- 白灼生菜/菜心/小白菜+金针菇（可加榨菜或海带丝），水煮虾
- 豆腐蒸虾仁，清炒西蓝花
- 什锦鸡蛋炒饭（胡萝卜丁、玉米粒、青豆、洋葱粒/可平时提前切好分包冰冻）
- 白灼蔬菜+ 香菇，1~2个煮鸡蛋
- 菠菜炒蛋
- 青瓜炒牛肉
- 酸奶泡水果麦片

### 优质脂肪
- 促进微量元素吸收，维持体内激素水平，作用不可忽视；
- 一般有酸奶、各类坚果、橄榄油等；一般作为加餐搭配食用

### 高质量水果
- 蓝莓（低糖富含花青素各种维生素）、火龙果/苹果（高纤维低卡）、柚子（低卡）、柠檬（低卡低糖）、西瓜（低热量，但一定要控量）
